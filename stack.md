# Stack Tecnológico de FormMaster

Este documento detalla el stack tecnológico utilizado en el proyecto FormMaster.

## Tecnologías Principales

*   **Framework Frontend:** [Next.js](https://nextjs.org/) (Utilizando el App Router)
    *   **Lenguaje:** TypeScript
*   **Librería UI:** [React](https://reactjs.org/)
*   **Estilos:**
    *   [React Bootstrap](https://react-bootstrap.github.io/) (Componentes UI)
    *   [Tailwind CSS](https://tailwindcss.com/) (Utilidades CSS)
*   **Base de Datos:** [PostgreSQL](https://www.postgresql.org/)
*   **ORM (Object-Relational Mapper):** [Prisma](https://www.prisma.io/)
*   **Autenticación:** Sistema de autenticación simple basado en contraseña para administradores.
*   **Iconografía:** [FontAwesome](https://fontawesome.com/)
*   **Notificaciones:** [Sonner](https://sonner.emilkowal.ski/)
*   **Generación de IDs Únicos:** [UUID](https://www.npmjs.com/package/uuid)
*   **Control de Versiones:** Git

## Entorno de Desarrollo

*   **Gestor de Paquetes:** npm
*   **Linter:** ESLint
*   **Transpilador/Compilador:** TypeScript
*   **Entorno de Ejecución:** Node.js

## Despliegue

*   **Plataforma:** Vercel (se infiere por la presencia de archivos de configuración `.vercel/`)
